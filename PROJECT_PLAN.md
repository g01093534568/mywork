# 개인 업무 관리 프로그램 (WorkLog)

## 프로젝트 개요
- 조직 목표와 개인 목표의 연계를 바탕으로 한 개인 성과관리 웹 애플리케이션
- 단일 HTML 파일 구조, Supabase 백엔드, Vercel 배포

---

## 주요 기능

### 1. 로그인
- 시설명 + 사원번호(6자리) + 패스워드 3가지 일치 시 로그인
- 패스워드 SHA-256 해시 저장 (레거시 평문 자동 마이그레이션)
- sessionStorage 세션 유지 (새로고침 시 자동 복원)
- 권한 구분: 관리자 / 시설관리자 / 사용자

### 2. 홈 / 목표 현황
- 조직 목표 체계: 미션 → 비전 → 전략목표 → 전략과제 도식화
- 시설 목표: 시설별 목표 및 KPI, 달성률
- 개인 성과 목표: 조직 목표 연계 개인 KPI 및 달성률

### 3. 할 일 목록
- 할 일 추가 / 수정 / 삭제 (모달)
- 분류(업무/개인/기타), 우선순위(높음/중간/낮음), 상태(미완료/완료)
- 시작일·마감일 설정, 마감 초과 시 강조 표시
- 반복 할 일: 요일 지정 반복, 당일 완료 체크
- 검색 및 다중 필터(상태·분류·우선순위)
- 데스크톱 테이블뷰 / 모바일 카드뷰 자동 전환
- 완료 시 시각적 취소선 처리

### 4. 업무 (업무일지)
- 날짜 이동(이전/다음) 및 당일 할 일 목록 조회·완료 처리
- 주요 활동 기록: 제목, 시간, 전략과제 연계, 주요 내용, 느낀점 회고
- 활동 추가 / 수정 / 삭제
- Google Calendar 연동: 월별 캘린더 뷰, 일정 팝업 확인 (OAuth 2.0, 자동 토큰 갱신)

### 5. 개인 (개인 관리)
#### 연간목표 탭
- 연간 목표 추가·수정·삭제
- 세부 목표(Target) 관리 및 달성률 추적

#### 증권&환율 탭
- 보유 주식 포트폴리오 관리 (현재가·수익률 자동 조회)
- 매매일지: 매수/매도 기록, 손익 계산
- 실시간 환율 (USD/KRW, JPY/KRW 등)

#### 운동 탭
- 골프: 라운딩 기록 (날짜·코스·스코어·메모), 연도별 그룹 2열 그리드
- 러닝: 러닝 기록, 대회 기록 (날짜·대회명·거리·기록)

### 6. 관리자
- **조직 목표**: 미션·비전·전략목표·전략과제 입력·수정
- **시설 목표**: 시설별 목표·KPI·달성률·전략과제 연계 관리
- **사용자 관리**: 사용자 추가·수정·삭제, CSV 업로드/다운로드
- **Google 계정**: Google Calendar OAuth 연동 설정

## 기술 스택

### Frontend
- 순수 HTML / CSS / JavaScript (단일 파일: `worklog-app.Html`)
- 반응형 디자인 (데스크톱·모바일)

### Backend / 데이터
- **Supabase** (PostgreSQL): 사용자·목표·할일·업무일지·운동·주식 등 전체 데이터
- 인메모리 캐시(`_C`) + sessionStorage stale-while-revalidate 캐시
- 로컬스토리지 → Supabase 1회 자동 마이그레이션

### 인증
- 자체 구현 (SHA-256 해시 패스워드, sessionStorage 세션)
- Google OAuth 2.0 (Calendar 연동)

### 배포
- **Vercel** (프로덕션: worklog-app-nine.vercel.app)
- GitHub 연동 (`g01093534568/mywork`, master 브랜치)
