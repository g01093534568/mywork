# 개인 업무 관리 프로그램

## 프로젝트 개요
- 조직 목표와 개인목표와의 연계
- 개인 성과관리, 효과적인 개인의 관리를 위한 웹 기반 애플리케이션

---

## 주요 기능

### 1. 로그인 기능: 3가지 조건이 일치할 경우 로그인 가능
- 시설명
- 사원번호(6자리)
- 패스워드
- 로그인 시 활기찬 하루를 시작하자는 의미의 한 줄 문구 출력(ai로 매일 자동 생성)

### 2. Main Page
- 조직 목표 출력: 미션-비전-전략목표-전략과제 도식화
- 개인 목표 출력: 조직 목표와 연계된 개인성과목표 및 KPI 노출(성과계획서) 바탕

### 3. 업무일지
- to do list(할 일 목록)
  - 할 일 추가/수정/삭제
  - 우선순위 설정(높음, 중간, 낮음)
  - 완료 상태 표시
  - 시작일, 마감일 설정
  - 완료 시 삭선: 완료 체크 시 당일에는 출력, 다음 날에는 미출력/마감일까지 미완료 시 완료될 때까지 출력
  - 검색 및 필터링
  
- 오늘 할 일 목록 
  - to do list 자료 불러옴
  - 오늘 주요 활동
    - 제목
    - 오늘의 주요활동과 전략과제와의 연계성 
    - 시간
    - 주요내용: 작성자 직접 작성
    - 추가, 수정, 삭제 기능
    - 느낀 점 회고: 작성자 직접 작성

### 4. 관리자 page
  - 관리자만 로그인 가능
  - 조직목표 입력, 수정, 삭제, 추가 -> Main Page 노출
  - 사용자 시설명, 사원번호, 패스워드 관리: 엑셀 파일로 업로드 및 다운로드 가능
  - 관리자는 사용자 추가, 수정, 삭제 가능
  - 권한 부여
    - 관리자: 앱 관리 전체 권한 부여
    - 시설관리자: 해당 시설에 대한 관리 권한 부여
    - 사용자: 앱 사용만 가능

## 기술 스택

### Frontend
- HTML/CSS/JavaScript
- 로컬 스토리지 또는 데이터베이스 연동

### Backend (필요시)
- Node.js/Express 또는 Python/Flask
- 데이터베이스: MongoDB/MySQL

### 배포
- GitHub Pages, Vercel, Netlify 등

## 사용 기술 노트

### 사용할 수 있는 옵션들:
1. **순수 HTML/CSS/JavaScript**: 가장 간단하고 빠름(가장 우선)
2. **React**: 재사용 가능한 컴포넌트 구축 가능
3. **Vue.js**: 중간 규모 프로젝트에 적합
4. **TypeScript**: 타입 안전성 추가

### 데이터 저장:
1. **구글 클라우드**(가장 우선)
2. **로컬 스토리지**: 브라우저에 저장 (간단함)
3. **IndexedDB**: 더 많은 데이터 저장 가능
